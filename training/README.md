# Training — 트랙 B (오프라인 R&D)

> [절대 규칙 E19] 이 디렉터리는 **서빙 요청 경로에 들이지 않는다.** 오프라인 분리.

## 목적
L4(`backend/app/corrections/`)가 모은 교정 라벨을 사용해 모델을 지도학습한다.

```
교정 데이터(L4) → datasets/ → 지도학습 → 모델 → backend stages 의 어댑터로 끼움
```

## 학습 대상 (난이도 순)
- `ocr/` — 한글 가사 OCR 지도학습. 가장 현실적 첫 타깃. MPS로 파인튜닝 가능.
- (장기) 밀집형 SATB 음표 OMR. 기호 단위 주석 필요, CUDA(외부 GPU) 권장.

## 상태
**자리만 잡은 빈 디렉터리.** 구현은 1단계/2단계 서빙 안정화 이후.
